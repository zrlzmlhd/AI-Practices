# 机器学习的通用流程
1. 定义问题：有那些数据可以使用 要解决什么问题 是否要手动添加label
2. 收集能够有效评估目标成功的方法 很多情况要找到相关领域的复杂指标 可以到论文里面找 或者一些论坛
3. 准备用于评估Model的验证过程 训练集 验证集 测试集 必须要定义 注意数据泄漏的问题
4. 数据向量化：将数据转换为模型可以处理的格式
5. 开发第一个Model 他必须打败一个基准的测试方法来表示至少model是有效的
6. 通过超参数优化等方法或者正则化来提升模型性能 我在“让模型性能发挥到极致”这个目录已经说明了
7. 注意进行超参数调整的时候小心测试集过拟合导致数据泄漏 所以保留一个单独的测试集
