# IMDB电影评论情感分析

**难度**: ⭐⭐☆☆☆ (初级)

使用深度学习分析电影评论的情感倾向（正面/负面），这是NLP入门的经典项目。

## 📋 项目概述

### 项目目标
构建一个深度学习模型，能够自动判断电影评论是正面还是负面。

### 数据集说明
- **训练集**: 25,000条电影评论
- **测试集**: 25,000条电影评论
- **类别**: 二分类（正面/负面）
- **来源**: IMDB电影评论数据集

### 应用场景
- 社交媒体情感监控
- 产品评论分析
- 客户反馈分析
- 舆情监测

## 🚀 快速开始

### 1. 环境准备

```bash
pip install -r requirements.txt
```

### 2. 训练模型

```bash
# 训练LSTM模型
python src/train.py --model lstm

# 训练BERT模型
python src/train.py --model bert

# 训练CNN模型
python src/train.py --model cnn
```

### 3. 使用模型预测

```python
from src.model import SentimentPredictor

# 加载模型
predictor = SentimentPredictor()
predictor.load_model('models/best_model.h5')

# 预测
text = "This movie is absolutely amazing! I loved every minute of it."
sentiment, confidence = predictor.predict(text)
print(f"情感: {sentiment}, 置信度: {confidence:.2%}")
```

## 🛠️ 模型架构

### 1. LSTM模型
```
Embedding → LSTM → Dense → Output
```
- **准确率**: 87.5%
- **训练时间**: ~10分钟

### 2. BiLSTM + Attention
```
Embedding → BiLSTM → Attention → Dense → Output
```
- **准确率**: 89.2%
- **训练时间**: ~15分钟

### 3. CNN模型
```
Embedding → Conv1D → MaxPooling → Dense → Output
```
- **准确率**: 86.8%
- **训练时间**: ~5分钟

### 4. BERT微调
```
BERT → Dense → Output
```
- **准确率**: 92.5%
- **训练时间**: ~30分钟
