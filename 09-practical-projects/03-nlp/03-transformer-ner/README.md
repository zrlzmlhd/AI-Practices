# 03_Transformer命名实体识别_中级

**难度**: ⭐⭐⭐☆☆ (中级)

## 📋 项目简介

本项目是一个中级级别的自然语言处理实战项目。

## 🎯 学习目标

- 掌握自然语言处理的基本原理
- 学习数据预处理技巧
- 实现完整的训练和评估流程
- 理解模型优化方法

## 📁 项目结构

```
03_Transformer命名实体识别_中级/
├── data/              # 数据目录
├── notebooks/         # Jupyter笔记本
├── src/              # 源代码
├── models/           # 保存的模型
├── results/          # 结果输出
├── README.md         # 项目说明
└── requirements.txt  # 依赖包
```

## 🚀 快速开始

### 环境配置

```bash
# 安装依赖
pip install -r requirements.txt
```

### 数据准备

数据集说明详见 `data/README.md`

### 训练模型

```bash
python src/train.py
```

### 评估模型

```bash
python src/evaluate.py
```
